The Automated Photometry of Transients pipeline (AUTOPHOT)

Brennan, S. J., & Fraser, M. · Astronomy & Astrophysics 667, A62 (2022)

arXiv abstract arXiv PDF Journal (DOI) Code (GitHub)

Summary

AUTOPHOT is a Python software pipeline for measuring the brightness of astronomical transients in imaging data. It is built for forced photometry: even when a source is faint or not cleanly detected, the code measures flux at the expected position using survey-style image stacks and careful treatment of noise. The paper documents the algorithms, uncertainty estimates, and practical workflow for time-domain follow-up.

Why it matters

Modern surveys discover thousands of transients per night; follow-up groups need tools that are fast, documented, and give consistent light curves across observers and telescopes. A published pipeline lets others reproduce your numbers, compare methods fairly, and build on the same foundation—standard practice in fields like particle physics, still growing in optical transient work. AUTOPHOT also supports several of the observational projects described elsewhere on this site.
