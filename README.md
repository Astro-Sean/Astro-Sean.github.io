# astro-sean.github.io

Personal site for **Seán Brennan** — static HTML and CSS, served with [GitHub Pages](https://pages.github.com/).

**Live site:** [https://astro-sean.github.io/](https://astro-sean.github.io/)

## Layout

| Path | Purpose |
|------|---------|
| `index.html`, `about*.html`, `code.html`, … | Main pages |
| `css/` | Stylesheets |
| `papers/` | First-author paper pages |
| `files/` | CV (PDF), photos, banner image, favicon |
| `robots.txt`, `sitemap.xml` | SEO |
| `.nojekyll` | Use GitHub Pages without Jekyll processing |

Optional: `images/` is copied to the published site if present.

## Editing and publishing

1. Edit HTML/CSS/assets here.
2. If you use a separate clone of this repo, run `sync-to-github.sh` from your working tree (see script comments for `--push`).
3. Push to `main` / `master` on GitHub; Pages updates in a minute or two.

No build step or package install is required.
